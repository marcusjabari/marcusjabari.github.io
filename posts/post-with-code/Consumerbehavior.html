1 Introduction

This project analyzes consumer behavior related to social media advertisements. The goal is to identify which demographic factors—specifically Age and Estimated Salary—lead to a product purchase. By applying both supervised and unsupervised machine learning, we aim to provide actionable insights for digital marketing strategies.

2 Data Overview

The dataset contains 400 observations of social media users.

  • Source: Kaggle (Social Network Ads)

  • Target Variable: Purchased (0 = No, 1 = Yes)

  • Predictors: Age (Numerical), EstimatedSalary (Numerical)

8 Unsupervised Learning Analysis

We use K-Means Clustering to find natural segments within the audience without looking at whether they purchased anything.

Code snippet

Code 
# Step 1: Scale the data (essential for K-Means)
df_scale <- df_clean %>% select(Age, EstimatedSalary) %>% scale()

# Step 2: Run K-Means with 3 clusters
set.seed(123)
km_res <- kmeans(df_scale, centers = 3, nstart = 25)

# Step 3: Visualize the Clusters
fviz_cluster(km_res, data = df_scale,
             palette = "jco",
             geom = "point",
             ellipse.type = "convex", 
             ggtheme = theme_minimal(),
             main = "Unsupervised Learning: Consumer Segmentation")

Insight: The unsupervised model identifies three distinct groups:

  1. Young/Low-Income consumers.

  2. Middle-Aged/Middle-Income consumers.

  3. High-Income/Older consumers.

10 Discussion & Significance

The Decision Tree (or KNN if you prefer) typically performs better here because the relationship between age, salary, and purchasing is not perfectly linear.

Practical Implications:

  • Primary Target: Users over the age of 45 are high-value targets regardless of salary.

  • Secondary Target: Users between 30-45 are likely to buy only if their salary exceeds a specific threshold (visible in the Decision Tree nodes).

  • Business Value: By using these models, a company can reduce ad spend by 30-40% by avoiding users in the “Young/Low-Income” cluster.

This analysis successfully demonstrated how data preparation, visualization, and machine learning can optimize marketing efforts. Future work could include adding more variables like “Time Spent on Site” or “Device Type” to increase predictive accuracy.
